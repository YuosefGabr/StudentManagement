Student Management System
نبذة عن المشروع

نظام إدارة الطلاب (Student Management System) هو تطبيق سطح مكتب تم تطويره باستخدام لغة Java بهدف تسهيل إدارة بيانات الطلاب داخل المؤسسات التعليمية. يوفر النظام واجهة رسومية سهلة الاستخدام تسمح بإدارة بيانات الطلاب بشكل منظم وسريع مع إمكانية حفظ واسترجاع البيانات من قاعدة البيانات.

فكرة المشروع

يهدف المشروع إلى تحويل عملية إدارة الطلاب من الأسلوب الورقي التقليدي إلى نظام إلكتروني يساهم في تنظيم البيانات وتقليل الأخطاء وتسهيل الوصول إلى المعلومات في أي وقت.

مميزات النظام
إضافة الطلاب الجدد إلى قاعدة البيانات.
تعديل بيانات الطلاب وتحديثها.
حذف بيانات الطلاب عند الحاجة.
البحث عن الطلاب بسهولة باستخدام أكثر من معيار.
عرض جميع بيانات الطلاب في جداول منظمة.
إدارة المعلومات الشخصية والأكاديمية للطلاب.
طباعة التقارير والبيانات المطلوبة.
ربط النظام بقاعدة بيانات MySQL لحفظ البيانات بشكل آمن.
واجهة مستخدم رسومية تم تطويرها باستخدام Java Swing.
التقنيات المستخدمة
Java
Java Swing
MySQL
JDBC
NetBeans IDE
آلية العمل

يعتمد النظام على قاعدة بيانات مركزية يتم من خلالها تخزين جميع معلومات الطلاب. عند تنفيذ أي عملية مثل الإضافة أو التعديل أو الحذف، يتم تحديث البيانات مباشرة داخل قاعدة البيانات لضمان الحفاظ على المعلومات وسهولة الوصول إليها.

أهداف المشروع
تسهيل إدارة بيانات الطلاب.
تقليل الوقت والجهد المبذول في العمليات الإدارية.
توفير وسيلة سريعة للبحث عن المعلومات واسترجاعها.
تحسين دقة البيانات وتقليل الأخطاء البشرية.
تطبيق مفاهيم البرمجة الكائنية (OOP) وقواعد البيانات في مشروع عملي متكامل.
## طريقة التشغيل

1. قم بإنشاء قاعدة بيانات MySQL واستيراد ملف قاعدة البيانات الخاص بالمشروع.
2. عدّل بيانات الاتصال بقاعدة البيانات داخل ملف `MyConnection.java`.
3. شغّل المشروع باستخدام NetBeans أو أي بيئة تطوير تدعم Java.
4. بعد تشغيل البرنامج ستظهر شاشة تسجيل الدخول.
